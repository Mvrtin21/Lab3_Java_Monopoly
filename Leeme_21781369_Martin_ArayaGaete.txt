CAPITALIA - Monopoly en Java (POO)
=====================================================================================================
CAPITALIA es una implementación en Java de un juego tipo Monopoly utilizando programación orientada a 
objetos. Permite crear partidas, gestionar jugadores, propiedades, construcción de casas/hoteles, 
cartas de suerte/comunidad, cárcel, hipotecas, renta, bancarrota y persistencia de partidas en JSON.

Uso del programa:

Al iniciar, el menú muestra opciones para:

- Crear nueva partida
- Importar partida desde archivo JSON
- Mostrar estado actual del juego
- Jugar turno completo
- Construir casa
- Construir hotel
- Hipotecar propiedad
- Deshipotecar propiedad
- Exportar partida a archivo JSON
- Pagar multa (si está en cárcel)
- Usar tarjeta comodín "Salir de la cárcel" (si aplica)
- Salir del juego

Persistencia de partidas:

- Exportar: elige la opción correspondiente e ingresa un nombre de archivo (ej. partida1.json). 
  El juego guarda el estado actual en JSON.

- Importar: desde el menú principal elige cargar partida e ingresa la ruta del archivo JSON 
  exportado. El juego se restaura al estado guardado.

Pruebas:
- Se incluyeron partidas ("martinEnCarcel.json"), digite textualmente el nombre de el archivo de 
  la partida exportada "martinEnCarcel.json"
  - Se encuentra un archivo llamado "Historial partida 'martinEnCarcel.json'" en la carpeta principal
    donde se refleja el historial de la partida, para entender como llego hasta su exportación.

Notas:
- Asegúrate de tener permisos de escritura en el directorio donde exportes JSON 
  (Que es la misma "CAPITALIA-CODIGO-FUENTE_21781369_ArayaGaete").
